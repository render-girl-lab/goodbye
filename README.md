# Última Carta — um adeus em pixels

Alguns adeus não cabem em palavras. Este tentou.

---

Este é um site pequeno demais para tanta saudade. Foi feito para ser aberto numa tela escura, sozinho, como quem relê uma mensagem que nunca deveria ter enviado.

Ele guarda uma carta que não foi respondida. Versos que demoram a aparecer, como memórias que a gente tenta não evocar. A chuva cai sem pressa. A música toca baixinho, quase se desculpando por ainda estar ali.

---

## O que há aqui

**A chuva** — porque algumas despedidas vêm com o céu.

**O poema** — que aparece devagar, verso por verso, como quem não quer chegar ao fim.

**A música** — uma canção antiga, guardada em `assets/audios/musica.mp3`. Ela sabe coisas que eu não consegui dizer.

---

## Como ler

Não há instruções. Só há um jeito de ler uma carta que não foi respondida: com o coração apertado e o volume baixo.

Abra `index.html`. Deixe carregar. Deixe a chuva cair. Deixe as palavras surgirem. Não tenha pressa. Quem está de partida não deve ser apressado.

---

## Sobre a música

Ela está ali para quem precisar. Se não precisar, a chuva sozinha já diz o bastante. Se precisar, aperte play. Ela não vai julgar.

---

Agora você pode fechar esta aba.

Ela não vai sentir falta.
